- 音源にAivisSpeechを使用するので、AivisSpeechのライセンスには従ってください。


## ■■ 対応機種 ■■

- Windows10以降(64bit)。グラフィックボードは特に必要ないです。
- 別途AivisSpeechのインストールが必要です。
- 長文対応。10万文字超えても問題ありません。
- 作者のPC(intel N97)の場合、3000文字（朗読12分ほど）の変換に大体10分でした。
- サンプルテキストとサンプル辞書が付属していますので、触ってみてください。


## ■■ 使い方 ■■

### 事前準備
AivisSpeechをインストールする。
- https://aivis-project.com/ ここからダウンロードしてインストール。
- インストール後、AivisSpeechを起動しておく（ポート10101で待機します）。

### プログラム実行
- zipを開いて、適当なフォルダに置く。
- 「日本語長文Audiobook Maker」を実行。

### 文章をM4Bファイルに変換
（入力ファイル設定）
- １．入力ファイル指定（txtまたはmd)
- ２．カスタム辞書ファイルあれば指定(json)

（オプション設定）
- ３．音声設定（Aivisのパラメータ）変えたければ指定。
- ４．ローカルAIのLMstudioがあるなら変換モード指定してもよい。プロンプトも指定できる。

（出力設定）
- ５．出力フォルダ指定。
- ６．M4B生成開始を押す→しばらく待つとm4bファイルが出てくる。


## ■■ オーディオブックの聴き方 ■■

- 出来あがったm4bファイルはWindows Media Playerでも再生できます。
- またAudibleや、iPhoneのBooksアプリなどを使うと、チャプター移動もできます。


## ■■ 入力小説テキストのフォーマット ■■
```sample
まえがき（作者名とか）
#章タイトル１
##話タイトル
・・・本文
##話タイトル
・・・本文
##話タイトル
・・・本文

#章タイトル２
##話タイトル
・・・本文
##話タイトル
・・・本文
・・・
```
という構成が可能です。各章・各話のサブタイの前に#または##を付けてください。


## ■■ カスタム辞書形式 ■■

- json形式で登録（サンプル参照。「"漢字": "よみ"」という形式）
- 長さが長いものが優先的に変換されます。
- カタカナは先頭アクセントになる
- スペースは半角です。またスペースを詰めたり崩さないでください。
